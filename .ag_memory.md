# [THE FOUNDATION]
- Core tech stack: React, Tailwind CSS, pdfjs-dist (v5.6.205)
- Routing: Component-based navigation (`PdfViewer`)
- Global state: React `useState` and `useRef` within `PdfViewer`. Annotation state in `annotationStore`.

# [THE FORGE]
- Feature: Advanced PDF/EPUB Viewing Experience Optimization. (COMPLETED)
- Files: `src/viewers/PdfViewer.tsx`, `src/index.css`
- Finalized Architectural Patch:
  1. **Continuous Virtualization:** Buffer-based page mounting (+/- 3 pages) with clean cleanup.
  2. **Coordinate Matrix Fix:** Switched to `convertToViewportPoint` for pixel-perfect overlays.
  3. **Fragmented Search:** Live strings joined across PDF text-items to handle split-word matches.
  4. **Selection Physics:** Solved "Darker Blocks" error by using opaque selection color + parent `opacity` + `mix-blend-mode: multiply`.
  5. **Visibility:** Fixed 0x0 size bug in single/two-page viewing modes.

# [THE GRAVEYARD]
- `TreeWalker` DOM mutation: Removed (Corrupted selection coordinate tracking).
- `findLastIndex` (ES2023): Replaced with manual for-loop due to target JS version.
- `pdfjsLib.Util.applyTransform`: Returns `void` in this version; switched to viewport methods.
- Transparent `::selection`: Caused additive alpha artifacts; switched to parent-transparency pattern.
- Buffer-less Scroll Tracking: Caused race conditions during fast scrolling; resolved via `renderPageRef`.

# [THE BLOCKERS]
- None. PDF viewer is currently at "GodMode" stability.
